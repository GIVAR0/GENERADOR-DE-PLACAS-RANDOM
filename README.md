<h1>Hola usuario, este es un pequeno proyecto para generar, mostrar placas aleatorias, mi objetivo es simplemente experimentar con datos generados al azar</h1>

<h2>TECNOLOGIAS USADAS</h2>
<ul>
  <li>Python version 3.10+</li>
</ul>

<h2>COMO USAR?</h2>

<ul>
  <li>Puedes clonar el repositorio.</li>
  <li>Entras al proyecto.</li>
  <li>Ejecutas el generador de placas.</li>
  <li>Presionas Enter.</li>
</ul>

<h2>COMO FUNCIONA EL CODIGO?</h2>

<ol>
  <li>Importacion de la libreria "random": se usa para generar numeros y caracteres al azar.</li>
  <li>Pausa inicial con input(), que hace que inicie el programa cuando presionas enter.</li>
  <li>Se imprimen líneas horizontales y un borde para simular la parte superior de la placa.</li>
  <li>El programa genera tres letras mayúsculas utilizando valores ASCII convertidos a caracteres.</li>
  <li>Se imprime un guion para poder separar la seccion de las letras de los numeros, y genera 2 numeros aleatorias y otra vez se pone otro guion, y se generan otros 2 numeros aleatorios y se imprimen dos lineas horizontales para simular la parte inferior de la placa.</li>
</ol>
